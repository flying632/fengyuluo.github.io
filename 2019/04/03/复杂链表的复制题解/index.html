复杂链表的复制题解

发表于 2019-04-03 | 分类于 算法题解

题目描述

输入一个复杂链表（每个节点中有节点值，以及两个指针，一个指向下一个节点，另一个特殊指针指向任意一个节点），返回结果为复制后复杂链表的head。（注意，输出结果中请不要返回参数中的节点引用，否则判题程序会直接返回空）

解题思路

  1. 先看清楚RandomListNode的结构组成，一个值元素，两个指针，一个指向next，一个随意指一个node，由此我们依据next指针可以遍历整个链表
  2. 我们遍历这个链表，每个节点node，我们都对应新建一个节点node1，插入到他的后面，这样可以保存lable和分辨出next指针
  3. 然后我们处理random指针，我们再遍历这个链表，每个节点node，将它的random指针指向节点的next节点赋值给node1节点，但是要注意Null pointer
  4. 分离原有链表和复制链表，注意变量的命名，有条理，要思路清楚
